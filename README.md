## 快速开始

```sh
# clone the project
git clone https://github.com/electron-vite/electron-vite-vue.git

# enter the project directory
cd electron-vite-vue

# install dependency
pnpm install

# develop
pnpm dev

# build
pnpm build
```

## 分支

`alpha`: 开发分支，用于开发环境

`beta`: 测试分支，用于测试环境

`main`: 主分支，用于生产环境
